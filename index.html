Creating something out of nothing

Here are some of the projects I've worked on, made with a lot of and .
Feel free to also visit my Github account.

Tribute - Biography

The goal of this project was to build a website that would be responsive across all platforms. Bootstrap 4 libary was used
for the making of it.

Taco Place Mock-Up Website

This project was made entirelly using CSS Grid for the layout. The challenge of it was to try to make it fully responsive while using as little media queries as possible.

Weather Now

Website application that displays the current weather of a destination that has been inputted by the user. Open Weather API was used to get the required data. Also, depending on the temprature displayed, the background image changes accordingly.

Random Quote Generator

The focus of this project was the use of jQuery and JSON in order to fetch random quotes with the forismatic API, and once the quote was loaded, the user should also have the ability to share the quote instantly on Twitter.

Memory Game

Second assignment for Udacity's Front End Developer Nanodegree Program. In this project, a memory game was created with 16 cards of 8 different images. The goal was to match two equal cards, have them stay open, and to continue this way until all cards have been matched together or until the user runs out of tries. Made with HTML, CSS and JQuery.

Move Planner App

The goal of this project was to utilize three different APIs as well as different AJAX calls. The first API used was Google Street View API to present street images depending on user's input destination. Secondly, the New York Times Articles API was used to show articles relevant to that destination. And lastly, Wikipedia API was used to present relevant links of that user input.
